# 📋 Guía del Usuario: Cómo Subir Documentos Pendientes de Fuentes de Pago

**Sistema**: RyR Constructora
**Fecha**: Diciembre 2025
**Versión**: 1.0

---

## 🎯 ¿Qué son los documentos pendientes?

Cuando se crea una **Fuente de Pago** (Crédito Hipotecario, Subsidio, Cuota Inicial, etc.), el sistema automáticamente genera una lista de **documentos requeridos** que deben subirse antes de poder **desembolsar** el dinero.

**Ejemplo:**
- Cliente: Juan Pérez
- Fuente de Pago: Crédito Hipotecario - Bancolombia - $50.000.000
- Documentos pendientes:
  - ✅ Carta de Aprobación (OBLIGATORIO)
  - ✅ Boleta de Registro (OBLIGATORIO)
  - ⚠️ Avalúo de Vivienda (OPCIONAL)

---

## 🚨 ¿Por qué es importante?

❌ **Sin documentos obligatorios** → El botón "Desembolsar" estará **BLOQUEADO**

✅ **Con todos los documentos** → Se puede **desembolsar** el crédito

---

## 📍 ¿Dónde veo los documentos pendientes?

### **En el Card de la Fuente de Pago (ÚNICO LUGAR)**

1. Entra al **detalle del cliente** (Juan Pérez)
2. Ve a la pestaña **"Vivienda Asignada"**
3. Busca la sección **"Fuentes de Pago"**
4. Dentro del card de cada fuente verás el progreso de documentos:

```
┌────────────────────────────────────────────────────────────┐
│ 💰 Crédito Hipotecario - Bancolombia                      │
│                                                            │
│ Progreso: ████░░░░ 2/5 pasos completados (40%)           │
│                                                            │
│ ❌ Carta Aprobación        [Subir documento]             │
│ ❌ Boleta de Registro      [Subir documento]             │
│ ✅ Certificado Tradición   [Ver PDF]                     │
│ ⚠️ Avalúo (opcional)       [Subir documento]             │
│                                                            │
│ ⚠️ Desembolsar [BLOQUEADO] - Faltan documentos           │
└────────────────────────────────────────────────────────────┘
```

---

## ✅ ¿Cómo subir un documento pendiente? (Paso a Paso)

### **Método ÚNICO: Desde el Card de la Fuente de Pago**

#### **Paso 1: Identificar el documento pendiente**

Busca el documento en el card de la fuente de pago. Por ejemplo:
```
┌────────────────────────────────────────────────────────────┐
│ 💰 Crédito Hipotecario - Bancolombia                      │
│                                                            │
│ ❌ Boleta de Registro          [📤 Subir documento]      │
└────────────────────────────────────────────────────────────┘
```

#### **Paso 2: Click en "📤 Subir documento"**

Al hacer click en el botón del documento específico, se abrirá automáticamente el formulario de subida con **TODO PRE-CONFIGURADO**:

```
┌────────────────────────────────────────────────────────────┐
│ 📤 Subir Documento                                         │
│                                                            │
│ ✨ Este documento validará automáticamente el requisito   │
│    "Boleta de Registro" del Crédito Hipotecario          │
│                                                            │
│ Categoría: [Certificados de Tradición ▼]  ← Pre-lleno    │
│ Título:    [Boleta de Registro - Crédito Hipotecario]    │
│                                                            │
│ ┌──────────────────────────────────────────────────────┐ │
│ │  Arrastra el archivo aquí                            │ │
│ │  o haz click para seleccionar                        │ │
│ └──────────────────────────────────────────────────────┘ │
│                                                            │
│ [Cancelar]  [Subir documento]                             │
└────────────────────────────────────────────────────────────┘
```

#### **Paso 3: Seleccionar el archivo**

- **Opción A**: Arrastra el archivo desde tu explorador de Windows
- **Opción B**: Haz click en la zona de carga y busca el archivo

#### **Paso 4: (Opcional) Personalizar el título**

Puedes cambiar el título sugerido si lo deseas:
```
Título: [Boleta de Registro - Bancolombia 2025]
```

#### **Paso 5: Click en "Subir documento"**

¡Listo! El sistema:
1. Sube el archivo
2. Lo vincula **automáticamente** con el requisito pendiente
3. Marca el paso como completado ✅
4. Actualiza el progreso (2/5 → 3/5)
5. Eli⚠️ IMPORTANTE: NO subir desde la pestaña Documentos (General)**

### **Método 3: Desde la pestaña Documentos (General)**

⚠️ **CUIDADO**: Este método **NO vincula automáticamente** el documento.

Si subes un documento desde el botón general "Subir documento" de la pestaña:

1. **NO se vinculará automáticamente** con el requisito
2. Tendrás que subirlo **de nuevo** desde el banner/card
3. **NO recomendado** para documentos de fuentes de pago

```
Usar solo para:
- Fotos de la vivienda
- Documentos generales
- Correspondencia
- Archivos sin validación automática
```

---

## 🎯 ¿Cómo sé que el documento se vinculó correctamente?

### **Señal 1: Banner azul de confirmación**

Después de subir, aparecerá un banner azul:
```
✅ Documento subido exitosamente y vinculado al requisito "Boleta de Registro"
```

### **Señal 2: El documento desaparece del banner amarillo**

Si tenías 3 documentos pendientes y subiste 1, ahora verás solo 2.

### **Señal 3: Progreso actualizado**

En el card de la fuente de pago verás:
```
Progreso: ████████░ 3/5 pasos completados (60%)
```

### **Señal 4: Icono verde (✅) en el requisito**

```
✅ Boleta de Registro      [Ver PDF]  ← Ahora muestra "Ver PDF"
```
Progreso actualizado en el card
⚠️ **ADVERTENCIA**: Si eliminas un documento que ya estaba vinculado:

1. El requisito vuelve a **incompleto** ❌
2. El documento vuelve a aparecer en el **banner amarillo**
3. El progreso **disminuye** (3/5 → 2/5)
4. El botón "Desembolsar" se **bloquea** nuevamente

**Solución**: Volver a subir el documento usando el banner amarillo.

## ❓ Preguntas Frecuentes

### **P: ¿Puedo subir cualquier archivo?**

R: El sistema acepta:
- ✅ PDF (recomendado)
- ✅ Imágenes (JPG, PNG)
- ✅ Word (DOC, DOCX)
- ✅ Excel (XLS, XLSX)
- ❌ Archivos ejecutables (.exe, .bat)
- ❌ Archivos > 50 MB

---

### **P: ¿Puedo cambiar el título del documento?**

R: ✅ **Sí**, puedes personalizar el título. El sistema sugiere uno, pero puedes cambiarlo.

**Importante**: No cambies la **Categoría** si viene pre-seleccionada (puede afectar la vinculación).

---

### **P: ¿Qué pasa si subo el documento equivocado?**

R: Puedes:
1. Ir a la pestaña **"Documentos"**
2. Buscar el documento subido
3. Click en **"Eliminar"**
4. Volver a subir el correcto desde el banner amarillo

---

### **P: ¿Puedo ver qué documentos ya están completos?**

R: ✅ **Sí**, en el card de la fuente de pago:
- ✅ Verde con PDF = Completado
- ❌ Rojo = Pendiente
- ⚠️ Amarillo = Opcional sin subir

---

### **P: ¿Cuánto tarda en vincularse el documento?**

R: **Inmediato** (menos de 1 segundo). El sistema usa triggers automáticos en la base de datos.

---

### **P: ¿Qué hago si el documento no se vincula?**

R: Verifica:
1. ¿Subiste desde el botón **"Subir ahora"** del banner?
2. ¿No modificaste la **Categoría** en el formulario?
3. ¿El archivo se subió correctamente (sin errores)?

Si el problema persiste, conta📤 Subir documento"** del card de la fuente
