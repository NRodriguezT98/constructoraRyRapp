## Versión: 3.4

## Reglas del Proyecto

- Strings inline máximo **80 caracteres**
- Si excede → extraer a archivo `.styles.ts`
- Dark mode obligatorio (`dark:*`)
- Glassmorphism: `backdrop-blur-xl bg-white/80`
- Gradientes de 3 colores: `from-X via-Y to-Z`
