## 🎯 Guías de Uso Rápido

### "Necesito implementar una feature similar"

**Ruta recomendada**:
1. Leer `SISTEMA-DESCUENTOS-VALOR-MINUTA.md` completo
2. Revisar código en `paso-1-info-basica-refactored.tsx`
3. Seguir patrón: Migración SQL → Tipos → Schema → Componente → Hook

---

### "Necesito agregar validación nueva"

**Ruta recomendada**:
1. Actualizar `asignar-vivienda.schema.ts` (Zod)
2. Si requiere BD: Crear trigger en migración SQL
3. Actualizar testing checklist con nuevo caso

---

### "Necesito modificar el diseño UI"

**Ruta recomendada**:
1. Leer `COMPONENTE-PASO1-DESCUENTOS-IMPLEMENTADO.md` - Sección "Implementación Técnica"
2. Revisar `DIAGRAMA-FLUJO-SISTEMA-DESCUENTOS.md` - "Estados Visuales"
3. Modificar secciones en `paso-1-info-basica-refactored.tsx`
4. Validar dark mode y responsive

---

### "Necesito crear reporte de descuentos"

**Ruta recomendada**:
1. Usar vista `vista_descuentos_aplicados`
2. Ejemplos en `SISTEMA-DESCUENTOS-VALOR-MINUTA.md` - Sección "Reportes"
3. Testing en `TESTING-CHECKLIST-SISTEMA-DESCUENTOS.md` - Fase 9
